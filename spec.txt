Спецификация для стойки:
Возвращает список свойств стойки/стеллажа в формате JSON:           
     {
        "room":"110",          /* Номер помещения                                                      */
        "name":"RA03",         /* Название серверного шкафа                                            */
        "type":"units",        /* Либо шкаф либо стеллаж: units (юнитовая стойка), towers (стеллаж)    */
        "size":48,             /* Размер в юнитах для units, 30 для стеллажа                           */
        "switch": [103,163]    /* Номера коммутаторов в стойке                                         */
     }
     
Спецификация для списка оборудования:
Возвращает список объектов. Каждый объект представляет собой отдельную единицу оборудования со своими свойствами.
[
    {
           "id":332,           /* Номер сервера в оборудовании                                         */
           "type":"unitserver",    /* 
                      Тип оборудования: 
                                    switch,            -- для коммутаторов
                                    unitserver,        -- для стоечных серверов
                                    bladechassis,      -- корзина для блейд-серверов
                                    fullbladeserver,   -- для цельных HP-блейдов
                                    halfbladeserver,   -- для сдвоенных HP-блейдов
                                    ucsbladeserver,    -- для Cisco-блейдов
                                    tower              -- для десктопных серверов
                                                                                                        */
           
           "size":1,           /*          
           Высота в юнитах для unitserver, switch, bladechassis. 
           Для остальных:
                                    fullbladeserver    -- 2
                                    halfbladeserver    -- 1
                                    ucsbladeserver     -- 1
                                    tower              -- 1
                                                                                                        */
           "units":[ 48 ],     /* Место установки сервера. Массив номеров юнитов в стойке для unitserver, switch, bladechassis.
           Для остальных (номер также возвращается в массиве):
                                    fullbladeserver    -- от 1 до 16
                                    halfbladeserver    -- от 1 до 16 с суфиксом 'a' или 'b'
                                    ucsbladeserver     -- от 1 до 8
                                    tower              -- от 1 до 30
                                                                                                        */
              
           "pdu":[
             /* Массив подключенных розеток                                                             */
              "102/2/3",
              "202/2/1"
           ],
           "maxdrive": 8,
      /* Вместимость дисков в сервер */
           "drives":[ /* Массив дисков, установленных в сервере, имеют 4 типа и объём			*/
             {"type": "sata-ssd", "capacity": "120G"}, /* Каждый диск представлен как объект            */
             {"type": "sata-hdd", "capacity": "1Tb"},
             {"type": "sas-ssd", "capacity": "2Tb"},
             {"type": "sas-hdd", "capacity": "146G"}
           ],
           "powerstate":1,     /* Состояние питания: 
                                    если заказ привязан и активный -- включен (1)
                                    если не привязян к заказу или заказ к удалению -- выключен (0)
                                    если занят нами -- включен (1)
           
                                                                                                        */
           "cross": [ /* Пока что не реализовано использование. В простейшей форме возвращает строку как в оборудовании, которую можно распарсить */
              "Ethernet185/1/8 Крос №00296 s263:p1",
              "Ethernet185/1/13 Крос №00301 s555:p1"
           ]
      }, // "id":332
]

Оборудование с типами bladechassis, fullbladeserver, halfbladeserver, ucsbladeserver дополнительно имеет поле class:
                                    bladechassis,      -- "class":"hp-c7000" или "cisco-ucs"
                                    fullbladeserver,   -- "class":"hp-c7000"
                                    halfbladeserver,   -- "class":"hp-c7000"
                                    ucsbladeserver,    -- "class":"cisco-ucs"
